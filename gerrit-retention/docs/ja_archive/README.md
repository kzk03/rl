# ドキュメント構成

このディレクトリには、Gerrit Retention IRLプロジェクトの詳細ドキュメントが含まれています。

## 📁 ディレクトリ構造

### 実験結果（experiment_results/）
モデル訓練と評価の実験結果レポート

- スライディングウィンドウ評価結果
- クロス評価実験
- パラメータ調整実験
- 精度改善実験

### 分析レポート（analysis_reports/）
データ分析と特徴量分析のレポート

- 特徴量重要度分析
- データセット統計
- モデル性能分析
- 予測カバレッジ分析

### 実装ガイド（implementation_guides/）
技術実装の詳細ガイド

- IRL実装ガイド
- 特徴量設計
- データ前処理
- 評価手法

### トラブルシューティング（troubleshooting/）
問題解決とデバッグ情報

- よくある問題と解決策
- データリークの修正
- パフォーマンス改善
- エラー対処法

### アーカイブ（ja_archive/）
過去の実験記録や古いドキュメント（参照用）
