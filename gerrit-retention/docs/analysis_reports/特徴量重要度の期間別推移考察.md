# 特徴量重要度の期間別推移考察

## 📊 概要

訓練期間（0-3m, 3-6m, 6-9m, 9-12m）ごとの特徴量重要度の推移を分析し、**時期によって継続予測に重要な要因がどう変化するか**を考察します。

---

## 🎯 主要な発見

### 1. 期間が進むにつれて重要度が減衰する特徴

#### **総レビュー数** 📉
```
0-3m: +0.0316 ⭐⭐⭐⭐
3-6m: +0.0176 ⭐⭐⭐
6-9m: +0.0102 ⭐⭐
9-12m: +0.0066 ⭐
```

**解釈**:
- 短期（0-3m）では**最も重要**だが、長期になるほど影響が減少
- **初期のレビュー経験**が継続を強く決定
- 長期的にはレビュー数だけでは継続を保証できない

**実務への示唆**:
- **新規参加者への集中投資**: 最初の3ヶ月が勝負
- 初期の20-30レビューが重要な閾値
- 長期的には他の要因（協力度など）が重要に

#### **総コミット数** 📉
```
0-3m: +0.0125 ⭐⭐⭐
3-6m: +0.0084 ⭐⭐
6-9m: +0.0071 ⭐
9-12m: +0.0038 ⭐
```

**解釈**:
- レビュー数と同様に減衰傾向
- 初期の開発活動が継続の基盤を作る
- 長期的には活動量より**活動の質**が重要に

#### **最近の活動頻度** 📉（特に注目！）
```
0-3m: +0.0174 ⭐⭐⭐
3-6m: +0.0121 ⭐⭐
6-9m: +0.0019 ⭐
9-12m: -0.0009 ❌（負に転じる！）
```

**解釈**:
- 短期では強い正の影響だが、長期では**負に転じる**
- 初期は「頻繁な活動」が重要だが、長期では「持続可能なペース」が重要
- 過度に頻繁な活動は燃え尽きのリスク？

**実務への示唆**:
- 初期3ヶ月: 頻繁な活動を推奨
- 長期（9-12ヶ月）: ペース配分、無理のない活動を推奨

### 2. 期間が進むにつれて重要度が増す特徴

#### **応答速度** 📈（大きな変化！）
```
0-3m: -0.0027 ❌（負の影響）
3-6m: +0.0035 ⭐
6-9m: +0.0048 ⭐
9-12m: +0.0052 ⭐⭐
```

**解釈**:
- 初期は素早い応答が**継続を阻害**（意外！）
- 長期になるほど応答速度が**重要**になる
- 初期は「じっくり考える時間」が必要、長期は「素早い対応」が評価される

**実務への示唆**:
- **新規参加者**: 応答速度を求めすぎない、丁寧なレビューを推奨
- **ベテラン**: 素早い応答が期待される、効率的なレビューフローを支援

### 3. 常に安定して重要な特徴

#### **協力度** 📊（最も安定）
```
0-3m: +0.0156 ⭐⭐⭐
3-6m: +0.0091 ⭐⭐
6-9m: +0.0131 ⭐⭐⭐
9-12m: +0.0146 ⭐⭐⭐
```

**解釈**:
- 全期間で**常に正の影響**
- 期間による変動はあるが、**一貫して重要**
- 3-6mで一時的に下がるが、6-9m以降で回復

**実務への示唆**:
- 協力文化は**全期間を通じて最重要**
- 継続的に協力的な行動を促進する仕組みが必要
- 最も投資すべき要因

#### **強度（ファイル数）** 📊
```
0-3m: +0.0101 ⭐⭐
3-6m: +0.0035 ⭐
6-9m: +0.0126 ⭐⭐⭐
9-12m: +0.0070 ⭐⭐
```

**解釈**:
- 6-9mで**ピークを迎える**
- 中期的に大規模なレビューが重要性を増す
- 初期と長期では重要度がやや低い

**実務への示唆**:
- **中期（6-9ヶ月）**: 有意義な大規模レビューを割り当てる好機
- 初期: 小規模から始める
- 長期: 規模より質を重視

### 4. 常に負の影響がある特徴

#### **平均活動間隔** 📉（常に負）
```
0-3m: -0.0165 ⚠️⚠️⚠️
3-6m: -0.0105 ⚠️⚠️
6-9m: -0.0067 ⚠️
9-12m: -0.0090 ⚠️⚠️
```

**解釈**:
- 全期間で**常に負の影響**
- 初期（0-3m）で最も影響が大きい
- 長期でも一定の負の影響が継続

**実務への示唆**:
- **全期間を通じて定期的な活動が重要**
- 初期の不規則な活動は特に危険
- 長期不在の早期検出が必要

#### **活動トレンド** 📉（常に負）
```
0-3m: -0.0046 ⚠️
3-6m: -0.0093 ⚠️⚠️
6-9m: -0.0051 ⚠️
9-12m: -0.0067 ⚠️
```

**解釈**:
- 減少傾向は**全期間で継続を阻害**
- 3-6mで影響が最も強い
- 活動が減少し始めたら要注意

**実務への示唆**:
- **活動減少の早期検出**: 特に3-6ヶ月目に注意
- トレンド分析による予防的介入
- 減少傾向を反転させる施策

#### **コード品質スコア** 📉（常に負）
```
0-3m: -0.0089 ⚠️⚠️
3-6m: -0.0106 ⚠️⚠️⚠️
6-9m: -0.0067 ⚠️
9-12m: -0.0052 ⚠️
```

**解釈**:
- 全期間で**負の影響**
- 3-6mで影響が最も強い（完璧主義による疲弊？）
- 一貫して高品質追求が継続を阻害

**実務への示唆**:
- **適度な品質基準**: 完璧を求めすぎない
- 特に3-6ヶ月目に燃え尽き注意
- 心理的安全性の確保

### 5. 符号が反転する特徴（興味深い！）

#### **レビュー負荷** 🔄
```
0-3m: +0.0105 ⭐⭐（正の影響）
3-6m: -0.0008 ❌（負に転じる）
6-9m: -0.0022 ⚠️
9-12m: -0.0033 ⚠️
```

**解釈**:
- 初期は適度な負荷が**継続を促進**
- 3ヶ月以降は負荷が**継続を阻害**
- 初期の責任感 → 長期の負担感

**実務への示唆**:
- **初期（0-3m）**: 適度なレビュー依頼で責任感を育成
- **中長期（3m以降）**: 負荷を軽減、過度な依頼を避ける
- 時期に応じた負荷管理が重要

#### **最近の受諾率** 🔄
```
0-3m: +0.0012 ⭐（わずかに正）
3-6m: -0.0032 ⚠️（負に転じる）
6-9m: -0.0007 ❌（わずかに負）
9-12m: +0.0023 ⭐（正に戻る）
```

**解釈**:
- 符号が**頻繁に変化**
- 受諾率と継続の関係は**複雑**
- 一貫した影響がない

**実務への示唆**:
- 受諾率を指標にするのは避ける
- 無理に受諾率を上げる必要はない
- 他の要因を優先すべき

## 🔍 興味深いパターン

### 1. U字型パターン（一時的に下がって回復）


- 3-6ヶ月は**調整期**で協力度が下がりやすい

- 時期に応じた課題の割り当てが重要

### 3. 単調減少パターン

## 📊 まとめ

### 期間別の優先施策

| 期間 | 最重要施策 | 警戒すべきリスク |
|------|-----------|----------------|
| **0-3m** | レビュー機会提供、定期的な活動 | 不規則な活動、孤立 |
| **3-6m** | 燃え尽き防止、負荷管理 | 完璧主義、過度な負担 |
| **6-9m** | 大規模レビュー割り当て、協力促進 | マンネリ、活動減少 |
| **9-12m** | 効率化支援、ベテランケア | 燃え尽き、離脱 |

### 一貫して重要な要因

1. **協力度** - 全期間で正の影響、文化として定着させる
2. **定期的な活動** - 不規則な活動は全期間で危険
3. **適度な品質基準** - 完璧主義は全期間で継続を阻害

### 実装の優先順位

1. **高優先**: 初期（0-3m）への集中投資
2. **高優先**: 協力文化の醸成（全期間）
3. **中優先**: 3-6ヶ月目の特別ケア
4. **中優先**: 期間別のオンボーディング
5. **低優先**: 受諾率の管理（影響が小さい）
